**Keeper** is a personal productivity app designed to help you stay accountable by organizing tasks, goals, and habits into one focused system.

## Tasks

Tasks are the core of Keeper. Each task carries **points**, structure, and intent.
- Smart, timely reminders tailored to your selected **Mentor Persona**.
- **Simple or Complex tasks**

  - One-off or lightweight habits, e.g., *Apply mascara*
  - or *Water the cactus every alternate Friday*

### Advanced Features

- Custom schedules (biweekly, monthly, etc.)
- Cron-like repetition for power users
- Tasks can be rescheduled

### Goals & motivation

Tasks can be tied to **Goals** for smarter reminders and motivation.

Examples:

- Task: *Solve 2 math problems*
  Goal: *Score above 80%*
- Task: *Save $10 every month*
  Goal: *Buy that hoodie*

### Time & scoring rules

- Repeating tasks can have:

  - Expiry windows
  - Prime-time periods
  - Bonus or penalty points based on timing

## Statistics

Keeper provides beautiful, visual insights into your progress

- Daily, monthly, and yearly trends
- Streak tracking
- Completion rates

### Mood tracking

- Mood sliders you can log anytime
- Mood vs productivity correlations

## Widget

Once set up, the **home screen widget** becomes the primary way to interact with Keeper.

Widget features:

- Daily progress bar
- Quick task actions
- Mood slider
- Dopamine hitting animations on task completion

## Export / Import

- All the data lives on your device. (subject to change)
- Easy import/export
- Export to:

  - Backup files
  - `.xlsx` (Excel) for visualizing your days and habits
